# Story 8.4: Set up Netlify Deployment for Frontend PWA

## Status: Draft

## Story
**As a** DevOps engineer,  
**I want** the Angular PWA deployed to Netlify,  
**so that** users can access the application.

## Acceptance Criteria
1. Netlify account and site created
2. GitHub repository connected
3. Build settings configured (npm run build:prod)
4. Environment variables set (API_BASE_URL)
5. Automatic HTTPS enabled
6. _redirects file for SPA routing
